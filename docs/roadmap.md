# Roadmap

## 2026 Q1-Q2: Prior Art + Foundation
- Publish architecture-level prior art (completed Feb 22, 2026).
- Finalize interface assumptions for bridge/runtime boundary.
- Define MVP validation matrix for inference workloads.

## 2026 Q3: MVP Milestone
- Bring up first proof-of-concept node pipeline.
- Validate memory-window switching behavior on representative models.
- Publish initial performance and stability notes.

## 2026 Q4: Pod Integration
- Compose multi-node pod topology.
- Validate orchestration, telemetry, and fault-handling paths.
- Release first reproducible deployment playbook (public subset).

## 2027 H1: Early Open Reference
- Publish broader design artifacts from validated MVP path.
- Expand contributor documentation and test harnesses.
- Start formal governance model discussions.

## Release policy
Implementation-sensitive artifacts are released in stages after validation,
with priority on reproducibility and safe disclosure.
